Overview:
--------
This is a very simple module to make it easier to edit (or simply see the tid) of terms.

By default it is available to everyone with administer taxonomy permissions.

Please, if you have any other ways you would like to make editing terms easier please contact me and I'll make you a co-maintainer so you can add it to this module.

Installation and configuration:
------------------------------
Please refer to the INSTALL file for complete installation and 
configuration instructions.

When enabled, A new tab will be seen on taxonomy/term/n pages,
and taxonomy/term/n/edit will provide 
[URL path settings] and [Menu settings] within the normal form.

Requires:
--------
 - Drupal 6.0
 - enabled taxonomy.module
 - optional : support for path.module
 - optional : support for image_gallery.module
