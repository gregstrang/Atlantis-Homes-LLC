INSTALLATION
------------
1.  Before installing the module you need to download and install the jQuery
    Cycle plugin. The plugin can be downloaded from:
    http://malsup.com/jquery/cycle/download.html. If you only want the fade
    transition download the "Cycle Core Only", otherwise for all the
    transitions download "Cycle Core With All Transitions".

2.  Place the jQuery Cycle plugin file in the image_cycle directory and rename
    the file to jquery.cycle.js.

3.  Enable the dependency modules: image_gallery and jquery_update.

4.  Enable image_cycle.

5.  See README.txt for more information on how to use the module.
